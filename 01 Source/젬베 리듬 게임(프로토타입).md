역할: 너는 HTML5 Canvas와 JavaScript 전문 웹 게임 개발자야.

목표: '젬베 리듬 게임' 웹앱을 만들어줘.
디자인 요구사항:화면 하단 중앙에 '젬베 드럼' 이미지가 있다고 가정하고 원형의 터치/판정 영역을 배치해줘.
판정 영역:좌측 (Slap): 날카로운 소리 담당 (키보드 'D' 키)
중앙 (Bass): 묵직한 소리 담당 (키보드 'Space' 키)
우측 (Tone): 맑은 소리 담당 (키보드 'K' 키)
노트 떨어지는 방식: 화면 위에서 아래로 노트(원형)가 떨어지며, 음악 박자에 맞춰 판정 라인에 왔을 때 키를 누르면 점수가 올라가게 해줘.
기능 요구사항:
AudioContext를 사용하여 소리 지연(latency)을 최소화해줘.
각 키를 눌렀을 때 시각적 피드백(버튼이 눌리는 애니메이션)을 줘.
점수(Score)와 콤보(Combo) 시스템을 넣어줘.
배경음악(BGM)이 끝나면 게임이 종료되고 결과창을 보여줘.
코드는 하나의 index.html 파일에 HTML, CSS, JS를 모두 포함해서 작성해줘 (또는 분리해도 됨).

참고: 이미지는 djembe_bg.png로, 사운드 파일은 bass.mp3, tone.mp3, slap.mp3가 있다고 가정하고 코드를 짜줘.

시각적 매핑: 업로드하신 이미지 위에 투명한 버튼을 씌워서, 키보드를 누를 때마다 이미지 속의 손바닥(Bass)이나 손가락(Tone/Slap) 부분에 빛이 나거나 파장이 퍼지는 애니메이션을 넣으면 타격감이 훨씬 좋아질 것입니다.
